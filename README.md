# Pine

Pine is an editor based on TUI, written in Rust.

This project *aims to* provide a clean and modern coding experience,
including LSP/DAP support, modern key mappings (both modal and non-modal), and so on.

> [!WARNING]
>
> This project is in alpha stage now and is not ready for production yet.

## Features and To-Dos

- [x] Text editing
- [x] Image viewing support ([supported formats](https://docs.rs/image/latest/image/codecs/index.html#supported-formats))
- [ ] Multi-buffer support
- [ ] LSP support
- [ ] DAP support
- [ ] Configuration
- [ ] Plugin system
- [ ] Client/server architecture

## Installation

This project hasn't been published to any package repositories yet.

### Compile from source

The only requirement is [the Rust toolchain](https://rust-lang.org/).

To run this editor, clone the repo and run `cargo run` inside.
